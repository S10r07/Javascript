"Number" (numero) es un tipo de datos en JavaScript que represeta datos numericos.
Ahora intentemos sumar 2 numeros usando JavaScript.

JavaScript utiliza el simbolo "+" como un operador de adicion cuando se coloca entre datos numeros.

Ejemplo:

        const myVar = 5 + 10;

"myVar" ahora tiene el valor de 15;
