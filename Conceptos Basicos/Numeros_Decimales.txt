También podemos almacenar números decimales en variables. Los números decimales a veces
se denominan números de coma flotante o flotantes.

Nota: No todos los números reales pueden representarse con precisión en coma flotante.
Esto puede llevar a errores de redondeo.

ACTIVIDAD: CREA UNA VARIABLE myDecimal Y DALE UN VALOR DECIMAL CON UNA PARTE FRACCIONAL.

        const ourDecimal = 5.7;
        const myDecimal = 2.3;

----------------------------
MILTIPLICACION DE 2 NUMEROS
----------------------------

En JavaScript, tambien puedes realizar calculos con numeros decimales, al igual que
con numeros enteros.

Multipliquemos dos numeros decimales para obtener su producto.

ACTIVIDAD: CAMBIA EL 0.0 PARA QUE EL PRODUCTO SEA IGUAL A 5.0.

        const product = 2.5 * 2.0;

----------------------------
DIVIDE UN DECIMAL ENTRE OTRO
-----------------------------

Ahora dividamos un decimal entre otro.

ACTIVIDAD: CAMBIA EL 0.0 PARA QUE EL COCIENTE(quotient) sea igual a 2.2.

        const quotient = 0.0 / 2.0; // Change this line
        const quotient = 4.4 / 2.0;
